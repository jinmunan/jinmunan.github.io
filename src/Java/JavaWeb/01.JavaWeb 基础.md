# JavaWeb 基础

## 网络通信三要素

- IP：计算机在网络中的唯一标识
- 端口：应用程序在网络中的唯一标识 (0~65535)
- 传输协议：`tcp:安全 三次握手 速度稍慢 udp:不安全 速度快`

## Tomcat

- Tomcat 是 web 服务器，存放项目的静态资源和动态资源
